# vue-blog
vue+vue-router+express+mongoose制作的简单博客程序（自用）

---

- [x] admin登录
- [x] 文章发布
- [x] 文章浏览
- [x] 文章修改删除
- [x] 文章批量删除
- [x] 查找文章
- [x] markdown
